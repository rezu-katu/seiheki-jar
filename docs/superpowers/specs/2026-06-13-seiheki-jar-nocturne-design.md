# 性癖の瓶 公開版デザイン仕様

## 目的

`site/` の静的アプリを公開版として仕上げる。現行の4列瓶グリッドは維持しつつ、全体を「夜の標本棚」方向に寄せ、瓶の塗りを水が注がれているような液体表現にする。

## デザイン方針

- 背景は暗い赤紫から黒に近い夜色。
- アクセントは淡い金色。チップ、見出し、選択状態に使う。
- 白いカードが並ぶ印象を減らし、必要な操作面だけを半透明の暗色パネルとして置く。
- 瓶セルの四角い囲いは表示しない。選択中だけ薄い発光や下線で示す。

## 瓶グリッド

- スマホ基準幅は390px前後。
- 瓶は横4列を維持する。
- 瓶の下のラベルは現行より大きく、太く、白系で目立たせる。
- 長いラベルは最大2行で収め、必要な場合だけ少し縮小する。

## 瓶の見た目

- 瓶は首、肩、胴が分かる丸みのあるシルエットにする。
- ガラスの輪郭は淡い金色寄りの線にする。
- 液体は直線の矩形塗りではなく、水面の波、明るい水面線、泡、ハイライトを重ねる。
- `NG` は瓶内に大きなグレーの×を表示する。

## 状態仕様

各瓶の状態は以下を持つ。

```js
{
  type: null | "do" | "recv" | "both" | "ng",
  doLevel: 1 | 2 | 3 | 4 | 5,
  recvLevel: 1 | 2 | 3 | 4 | 5
}
```

- `do` は青い液体を `doLevel` の水位で表示する。
- `recv` は赤い液体を `recvLevel` の水位で表示する。
- `both` は左半分を青、右半分を赤にして、`doLevel` と `recvLevel` を別々の水位で表示する。
- `ng` は水位を使わず×を表示する。
- `clear` は `type` を `null` に戻し、両レベルを3に戻す。

## ボトムシート

- 種別ボタンは `したい`、`されたい`、`両方`、`NG`、`クリア`。
- `したい` 選択時は `したい強さ` の1行だけ表示する。
- `されたい` 選択時は `されたい強さ` の1行だけ表示する。
- `両方` 選択時は `したい強さ` と `されたい強さ` の2行を表示する。
- 未選択または `NG` の時は強さ行を非表示にする。

## 生成画像

- 生成画像にも同じ「夜の標本棚」方向を反映する。
- 画像は既存仕様を踏襲し、1080px幅の縦長PNGにする。
- 4列瓶グリッド、目立つラベル、液体水位、左右別水位を反映する。
- アプリ内SVGとCanvas生成画像の瓶ロジックはできるだけ同じ考え方で揃える。

## 公開

- 静的サイトとして公開する。
- 公開URL確定後、`SITE_URL` と `og:url` を差し替える。
- 公開後はアプリ本体、主要アセット、生成操作を確認する。
